img-taggen

Bild-taggen används med src som är länk till bilden, alt är det som ska läsas upp för skärmläsare, höjd och bredd kan också defineras. I CSS så kan man sätta regler för bilden till exempel att den ska vara både genomskinlig som bilden till vänster och jag har också via css filter på vilden i h1 lagt en gråskala-filter över den.

Med hjälp av clear så flyttar jag ner denna raden till ny rad.
